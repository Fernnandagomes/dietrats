# DietRats

O DietRats é um aplicativo de registro de dietas com foco em motivação social. A plataforma combina acompanhamento alimentar com competição entre amigos, incentivando os usuários a manterem consistência com hábitos saudáveis.

A proposta do aplicativo é transformar o processo de seguir uma dieta em uma experiência social (quase como rede social), competitiva e motivadora.

### Valor principal do produto
Ao acompanhar o progresso de amigos e participar de rankings dentro de grupos, os usuários tendem a manter maior disciplina e engajamento no plano alimentar.

### Principais funcionalidades
- Registro rápido de refeições:
    Permite acompanhar a alimentação diária de forma simples.
- Grupos de amigos
    Usuários podem criar grupos privados para acompanhar o progresso coletivo.
- Ranking e competição saudável
    Sistema de pontuação que incentiva consistência e disciplina na dieta.
- Feed social
Atualizações e registros compartilhados entre membros do grupo.

### 
**Projeto desenvolvido para a disciplina de NoSQL, utilizando banco de dados não relacional para armazenar dados de usuários, refeições, grupos e interações sociais.**
